INFORMACIÓN METODOLÓGICA


*Descripción de los métodos utilizados para la recolección/generación de datos: El diseño experimental se basó en la solicitud de bases de datos en los sistemas públicos, relacionados a embarazos en las localidades que integran la población de estudio, a partir de allí se solicitaron entrevistas en profundidad a quienes conformaron la muestra a incluir el análisis.  
*Métodos para el procesamiento de datos:  El procesamiento fue mediante la ejecución de algoritmos y manipulación de los datos.
*Información específica del instrumento o software necesarios para interpretar los datos: El análisis de los datos fue a través de los software estadístico infostat/E tras la generación de estadísticos con sus correspondientes tablas y gráficos y software Atlas.Ti 2025. Versión 25.0.1.32924 donde se generaron códigos, memos, redes y asociaciones entre ellos representados en tablas y sus correspondientes gráficos.
